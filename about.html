Tentang Website Ini / Tentang Saya

Tujuan Website

Website ini dibuat sebagai bagian dari tugas mata kuliah Pemrograman Web 1 di Universitas Pamulang. Tujuannya adalah untuk mempraktikkan penggunaan hyperlink dalam HTML dan dasar-dasar pengembangan web.

Selain itu, website ini juga berfungsi sebagai platform sederhana untuk berbagi informasi tentang (sebutkan topik kegemaran , misalnya: hobi saya, minat saya di bidang teknologi, atau sekadar perkenalan diri).

Tentang Pembuat

Saya AlI MUSTOFA, seorang mahasiswa/i (TEKNIK INFORMATIKA) di Universitas Pamulang. Saya tertarik di bidang yang saya minati, pengembangan web, desain grafis, data science.

Melalui website ini, saya berharap dapat terus belajar dan mengembangkan keterampilan saya di dunia pemrograman web.
